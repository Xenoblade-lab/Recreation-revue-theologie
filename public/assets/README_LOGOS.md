# Logos des moyens de paiement

Pour que les logos s'affichent correctement dans le popup d'abonnement, veuillez placer les 3 images suivantes dans ce dossier (`public/assets/`) :

1. **orange_money.png** - Logo Orange Money
2. **mpesa.png** - Logo M-Pesa (Vodacom)
3. **airtel_money.png** - Logo Airtel Money

Les images doivent être nommées exactement comme indiqué ci-dessus.

Si les logos ne sont pas trouvés, des placeholders textuels s'afficheront automatiquement.
